# 数学小博士 - 网页版

## 🎯 项目简介

**数学小博士**是一个专为三年级学生设计的数学考试与学习系统网页版。基于北师大版教材，提供智能题库、错题分析、学习报告等功能。

## ✨ 核心功能

### 1. 用户系统
- 学生登录/注册（本地存储）
- 个人信息管理
- 学习记录保存

### 2. 题库系统
- 北师大版三年级数学题库
- 多种题型支持（选择题为主）
- 智能题目推荐

### 3. 考试功能
- 模拟真实考试环境
- 计时功能
- 自动评分

### 4. 学习分析
- 学习进度跟踪
- 错题本功能
- 成绩统计图表
- 个性化学习建议

### 5. 报告系统
- 学习报告生成
- 知识点掌握分析
- 进步趋势图表

## 🔧 开发指南

### 添加新题目
在 `js/app.js` 的 `initQuestionBank` 函数中添加：

```javascript
{
    id: 6,
    type: 'single_choice',
    content: '题目内容',
    options: [
        { id: 'A', text: '选项A' },
        { id: 'B', text: '选项B' },
        { id: 'C', text: '选项C' },
        { id: 'D', text: '选项D' }
    ],
    correctAnswer: 'A',
    explanation: '题目解析',
    knowledgePoints: ['知识点1', '知识点2'],
    difficulty: 'easy', // easy, medium, hard
    estimatedTime: 30 // 预计答题时间（秒）
}
```

### 自定义样式
- 修改 `css/style.css`
- 主题颜色在 `:root` 变量中定义
- 响应式断点：768px（平板）、480px（手机）

### 扩展功能
1. **添加新页面**：创建新的HTML文件
2. **添加新功能**：在 `js/` 目录创建新模块
3. **集成后端**：可添加API调用
